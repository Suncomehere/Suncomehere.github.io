hexo博客默认主题添加多说评论

自从前段时间搭建了hexo博客后，一直想增加评论区上去，没评论区这博客写的多没意思啊，所以想着法儿的找办法。从网上也看了很多博主的方法，看的是我一头懵，试来试去不成功，今天算是终于加上去了。所以为了那些和我同样处境的朋友们，在这里就详细介绍下这评论是怎么加上去的，其实成功了之后才发现超简单。

*hexo默认主题添加多说评论

在这里我为什么讲默认主题landscape呢，因为我就是用的这个主题，其他的主题也没有特别喜欢的。所以给默认的主题添加一些需要的东西变得很迫切，这先讲添加评论，下一篇说一下怎么添加自己的头像。

1.登录多说评论,得到你的shortname.

首先你要有一个多说的二级域名，就是shortname,应该怎么得到它呢？

— 登录多说，用qq或者什么账号你自己看着办，之后顺着来你会到下面这个页面：

黑色圈住的部分就是你的shortname,接下来要用到的，你要记住。

如果有些小盆友已经登录过但是又没记住这个，没关系，你可以在后台管理的设置中看到它。

2.打开你的博客文件夹根目录下的_config.yml文件

– 在该文件里添加duoshuo_shortname: 你的shortname

这里就用到了上一步得到的shortname了，冒号后要有空格，放在文件的什么位置都可以，为了方便理解，你最好加在theme的下边一行吧，之后保存。

3.打开你的themes/landscape/layout/_partial/article.ejs文件

在这个文件里边最下面有一段代码：

<% if (!index && post.comments && config.disqus_shortname){ %>
 <section id="comments">
 <div id="disqus_thread">
<noscript>Please enable JavaScript to view the <a href="//disqus.com/?     ref_noscript">comments powered by Disqus.</a></noscript>
</div>
</section>
<% } %>

替换成：

  <% if (!index && post.comments && config.duoshuo_shortname){ %>
  <section id="comments">
 <!-- 多说评论框 start -->
 <div id="ds-thread" class="ds-thread" data-thread-key="<%= post.path %>" data-title="<%= post.title     %>" data-url="<%= post.permalink %>"></div>
 <!-- 多说评论框 end -->
 <!-- 多说公共JS代码 start (一个网页只需插入一次) -->
  <script type="text/javascript">
    var duoshuoQuery = {short_name:"<%= config.duoshuo_shortname %>"};
(function() {
    var ds = document.createElement('script');
    ds.type = 'text/javascript';ds.async = true;
    ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
    ds.charset = 'UTF-8';
    (document.getElementsByTagName('head')[0] 
     || document.getElementsByTagName('body')[0]).appendChild(ds);
})();
</script>
 <!-- 多说公共JS代码 end -->
  </section>
  <% } %>

什么都不用改，就直接替换就好了。之前让我觉得懵的就是看其他博文的这段代码有几个参数要自己填写，不知道填什么，时间都浪费在这段代码上了。

4.执行两个命令

先执行：

$ hexo g

后执行：

$ hexo d

到这之后你再运行一下你的博客，应该评论就加上去了，不过首页是没有的，要打开你的文章后下边就有了。

好了，也是时候让你的伙伴给你的文章吐槽一下了。
