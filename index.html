gulp的简单使用

学习了gulp发现，工作中用第三方工具的好处真的很多啊，今天就简单来讲一下gulp这个工具，概念性的东西在这不多说，直接运用起来。gulp是基于任务的自动构建工具，通过插件配合解决页面压缩、JS合并、CSS文件压缩等等功能，简单又强大。

一、gulp安装

* 先安装node.js

gulp是基于Node.js的，所以你需要先安装Node.js,下载安装后，

在终端命令执行：

$ node -v

如果结果出来node版本号，就说明安装成功。

* 创建项目文件夹，执行安装gulp命令

· 新建一个文件夹，例如test,在此文件夹下执行以下命令：

$ npm install -g  gulp    //这是全局安装，去掉-g后就是局部安装。
