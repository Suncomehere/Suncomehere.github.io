Hexo博客搭建

1.Hexo介绍

Hexo是一个基于Node的静态博客框架。通过它我们可以快速创建自己的博客，只需要几条命令即可。发布博客，这里讲解的时部署在github上面，不仅省去服务器成本，还可以减少很多麻烦事。接下来会介绍Hexo静态博客的搭建和发布方法。

2.Hexo安装

安装Hexo之前，你需要知道准备什么。

  • 下载安装 Node.js。Node.js下载地址

    这个下载很容易，直接进入下载地址，点击下载建议版本就OK了。

  • 下载安装 npm。npm下载地址

    npm是一个nodeJS包管理和分发工具，可以用终端命令来安装

    $ npm install gulp
  • 安装Git。(安装Xcode就自带有Git了)
  • 申请GitHub。
