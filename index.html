SEO搜索引擎优化

搜索排名问题相信是每个网站都希望做好的问题。在这里给大家介绍一些关于SEO搜索引擎优化的基础知识。

那有哪些东西可以影响网站的排名呢？

1.首先是title,就是一个网页的标题。

在一个网页的代码基础结构中有title标签,要想提高排名，标题中包含关键词的作用不言而喻，就像一篇文章主要写的是什么可以从文章标题中看出来的。

2.另外一个重要的标签就是meta标签。

从上面代码中可以看出网页的基础代码也包含meta这个标签，那它在SEO上又有怎样的作用呢？接下来我们看一下meta标签的一些属性。

  • keywords 关键词

    这个属性相信明眼人一看就懂，当你想搜什么东西的时候输入的不就是关键词嘛，那关键词写的越多越好吗？当然···不是，什么事都有一个度，最好设置10个左右。

    总会有些人自以为聪明，投机取巧的把关键词设置的和文章内容差的不止有十万八千里，那怎么办，搜索引擎就被他们玩弄了吗？

    NO,为了不让这些人得逞，所以这个属性并不起决定性作用，只是有一定的参考价值。

  • decription 描述

    这个属性在SEO上算是起到比较重要的作用，写描述语句的时候尽量包含多的关键词，写的好不好看你的语文功底啦。

3.有强调语义的标签

有强调作用的一些标签在搜索引擎优化上也有一定的作用。

  • h1~h6标签–标题类标签
  • strong标签–字体加粗

  • em标签–斜体

那有人说啦，“我把文章里的关键字都用强调标签包裹起来”，别做梦啦，规定整个网页中对于关键词的强调不能超过7次。

好啦，关于SEO搜索引擎优化的基础知识到此为止了，这里只是浅谈，你想深究，请转场地。

Hexo博客搭建

1.Hexo介绍

Hexo是一个基于Node的静态博客框架。通过它我们可以快速创建自己的博客，只需要几条命令即可。发布博客，这里讲解的时部署在github上面，不仅省去服务器成本，还可以减少很多麻烦事。接下来会介绍Hexo静态博客的搭建和发布方法。

2.Hexo安装

安装Hexo之前，你需要知道准备什么。

  • 下载安装 Node.js。Node.js下载地址

    这个下载很容易，直接进入下载地址，点击下载建议版本就OK了。

  • 下载安装 npm。

    npm是一个nodeJS包管理和分发工具，可以用终端命令来安装

    $ npm install gulp
  • 安装Git。(安装Xcode就自带有Git了)
  • 申请GitHub。

3.Hexo搭建步骤

环境和工具都准备好了之后，开始搭建博客。

1.新建一个文件夹用于存放相关搭建博客的文件，例如blog。

2.在终端中执行以下命令 –安装hexo
